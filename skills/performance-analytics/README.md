# Performance Analytics

On-demand marketing performance analysis.

## What It Does

Pulls data from Google Analytics (GA4 MCP), Ahrefs, Slack, web search, and content logs to produce actionable insights about what's working and what's not.

## Data Sources

- **GA4 MCP:** Real Google Analytics data (traffic, top pages, sources, trends)
- **Ahrefs MCP:** SEO rankings, keyword positions, domain metrics
- **Content logs:** What was published, when, on which platform
- **Experiment tracker:** Active experiments and their results

## Output

Structured performance report with:
- Week-over-week and month-over-month comparisons
- Channel-level breakdown (organic, social, direct, referral)
- Content effectiveness analysis
- Actionable recommendations
