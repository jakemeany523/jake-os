# CI Brief Rubric

## Dimensions

### 1. signal_density
How many non-obvious changes surfaced vs last scan? 0-1 = 3. 2-3 = 6. 4+ with explanations = 9.

### 2. sourcing
Every claim tied to a URL or dated artifact. Unsourced = 2.0 cap.

### 3. strategic_implication
Each finding answers "so what for LayerLens?" Descriptive-only = 4. Action-ready = 9.

### 4. delta_clarity
Compares explicitly to the last scan (memory/competitive-intel/briefs/). No delta = 5.0 cap.

### 5. brevity
≤1 page, scannable, the "5-minute read" bar. Fluff = 4.

### 6. action_items
Ends with 1-3 concrete action items Jake can decide on. Missing = 5.0 cap.

## Hard caps

- Unsourced claim = 2.0 max.
- No delta vs prior scan = 5.0 max.
- More than 1 page = 6.0 max.
