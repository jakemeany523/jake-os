# Blog Production Pipeline

Full blog workflow from research through CMS publishing.

## What It Does

Bridges research, writing, SEO/AEO optimization, quality control, and CMS insertion into a single flow. Handles the full lifecycle from topic selection to Framer CMS draft creation.

## Workflow

1. Topic research and angle development
2. Article drafting with SEO and AEO optimization
3. Copy Review pass for brand voice compliance
4. Image and graphic generation
5. Framer CMS formatting and insertion
6. Structured data schema generation

## Integration Points

- **Ahrefs MCP:** SEO research
- **Copy Reviewer:** Quality control
- **Framer CMS:** Publishing pipeline
- **Branded Graphics:** Image generation with brand identity
