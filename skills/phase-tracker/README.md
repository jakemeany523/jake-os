# Phase Tracker

90-day plan progress monitoring.

## What It Does

Checks progress against the current 90-Day Plan, tracking phase completion, milestone status, and weekly focus alignment. Identifies what's done, what's behind schedule, and what needs attention to maintain on-pace execution.

## Output

- Phase completion percentage
- Milestone status (done, in progress, behind, blocked)
- Weekly focus alignment check
- Recommended priority adjustments
