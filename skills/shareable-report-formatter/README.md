# Shareable Report Formatter

Report formatting for stakeholder distribution.

## What It Does

Transforms raw report output (weekly performance briefs, monthly reviews, ad-hoc analysis) into polished, shareable documents suitable for leadership, team, or external stakeholder distribution.

## Supported Formats

- PDF reports with branded formatting
- Slide decks for presentations
- Formatted documents for email distribution

## When It's Used

- Weekly performance briefs going to leadership
- Monthly strategy reviews
- Ad-hoc analysis that needs to be shared outside the immediate team
