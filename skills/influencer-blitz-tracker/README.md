# Influencer Blitz Tracker

Campaign tracking for influencer marketing initiatives.

## What It Does

Tracks influencer campaigns across outreach status, creator responses, deliverables, spend tracking, and ROI metrics. Maintains a persistent tracker file with status for each creator, newsletter partner, and amplification source.

## What It Tracks

- Outreach status per creator (contacted, responded, negotiating, confirmed, live)
- Deliverable status (drafted, reviewed, published)
- Spend vs. budget per partner
- ROI metrics when measurable (traffic, signups, mentions)
