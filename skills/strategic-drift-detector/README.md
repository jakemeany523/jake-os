# Strategic Drift Detector

Self-correction system that catches inconsistencies across the operating system.

## What It Does

Detects semantic drift when skills, scheduled tasks, CLAUDE.md, and content files fall out of sync with source-of-truth memory context files. Catches stale differentiators, outdated model counts, misaligned competitive framing, and product positioning changes before they compound.

## What It Checks

- Skills referencing outdated product positioning
- Content using stale competitor framing
- Scheduled tasks with misaligned context
- Model counts or benchmark references that are no longer current
- Messaging that contradicts recent strategic decisions

## Why This Matters

In a system with 16 skills, 11 agents, and 50+ memory files, drift is inevitable. A positioning change in `context/company.md` needs to propagate to every skill that references it. This skill catches the gaps before they become visible in published content.
