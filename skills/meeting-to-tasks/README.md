# Meeting to Tasks

Fireflies transcript to Notion task pipeline.

## What It Does

Converts meeting action items into real Notion tasks automatically. Parses Fireflies meeting summaries, identifies action items, and creates tasks with Status: Inbox so they surface in the daily task review.

## How It's Triggered

- **Automatically** by Morning Commander (daily) and Weekly Architect (weekly)
- **Manually** for any specified date range

## Why This Matters

Before this skill, meeting action items either lived in a Fireflies summary no one checked or got mentioned in a Slack brief that got buried. Now every meeting commitment lands in the Notion task queue automatically.

## Integration Points

- **Fireflies MCP:** Meeting transcripts and action items
- **Notion:** Task creation with proper status and categorization
