# Content Repurposer

Multi-platform content adaptation from a single source asset.

## What It Does

Takes one content asset (blog article, benchmark data, evaluation result, report) and produces native content for each distribution channel: Twitter threads, LinkedIn posts, newsletter excerpts, and social snippets. Each platform gets content tailored to that medium, not a copy-pasted reformat.

## Workflow

1. Analyze source content for key insights and angles
2. Generate Twitter/X version (thread or single post)
3. Generate LinkedIn version (educational framing)
4. Generate newsletter excerpt
5. Generate social graphic brief
6. Run all outputs through Copy Reviewer

## Why This Matters

One piece of research or writing can produce 4-5 distribution touchpoints. This is the biggest time multiplier in the content stack.
